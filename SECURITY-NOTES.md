# Security Notes

PM道場は静的ファイルだけで動くローカル/Pages向けアプリです。

## GitHub Pagesで公開してよいもの

- `index.html`
- `styles.css`
- `app.js`
- `data/*.js`
- `README.md`
- `.nojekyll`

## 公開しないもの

- 実案件名
- 勤務先名
- 顧客名
- 個人情報
- 実際のプロジェクト文書
- スクリーンショットや一時ファイル

## 保存データについて

進捗、連続学習、弱点、論文メモ、架空プロジェクト設定はブラウザの`localStorage`に保存されます。GitHub Pagesへアップロードされません。

ただし、アプリ内の初期データとして`data/*.js`や`app.js`に書いた内容は公開されます。公開して困る内容はコードやデータファイルに入れないでください。
